# سعيد أحمد - بورتفوليو محترف مونتاج الريلز والفيديوهات

## 🎬 نظرة عامة
موقع بورتفوليو احترافي لعرض أعمال سعيد أحمد في مجال مونتاج الريلز والفيديوهات.

## ✨ المميزات
- تصميم عصري ومتجاوب مع جميع الأجهزة
- أنيميشنز سلسة وجذابة
- محسّن لمحركات البحث (SEO)
- سرعة تحميل عالية
- تأثيرات بصرية احترافية
- نموذج تواصل فعّال
- روابط السوشيال ميديا

## 🚀 التقنيات المستخدمة
- HTML5
- CSS3 (مع Animations متقدمة)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Animate.css
- Google Fonts (Cairo)

## 📱 الأقسام
1. **الصفحة الرئيسية**: عرض تقديمي جذاب مع أنيميشنز
2. **عني**: معلومات عن سعيد والمهارات
3. **أعمالي**: معرض الأعمال والمشاريع
4. **تواصل معي**: نموذج تواصل ومعلومات الاتصال

## 🎨 التخصيص

### تغيير المعلومات الشخصية
افتح `index.html` وعدّل:
- معلومات التواصل (البريد والهاتف)
- روابط السوشيال ميديا
- النصوص والمحتوى

### تغيير الألوان
افتح `style.css` وعدّل المتغيرات في `:root`:
```css
--primary-color: #6366f1;
--secondary-color: #8b5cf6;
```

### إضافة أعمال جديدة
في `index.html`، أضف عنصر جديد في `.portfolio-grid`:
```html
<article class="portfolio-item fade-in-element">
    <div class="portfolio-image">
        <i class="fas fa-play-circle"></i>
        <div class="portfolio-overlay">
            <span>شاهد الآن</span>
        </div>
    </div>
    <div class="portfolio-info">
        <h3>عنوان العمل</h3>
        <p>وصف العمل</p>
    </div>
</article>
```

## 🔧 التثبيت والاستخدام

1. حمّل جميع الملفات
2. **مهم**: اتبع خطوات إعداد الفورم في ملف `formspree-setup.md` (3 دقائق فقط)
3. افتح `index.html` في المتصفح
4. أو ارفع الملفات على استضافة ويب

## 📊 تحسين SEO

الموقع محسّن بالكامل لمحركات البحث:
- ✅ Meta Tags كاملة
- ✅ Open Graph للسوشيال ميديا
- ✅ Schema.org Structured Data
- ✅ Sitemap.xml
- ✅ Robots.txt
- ✅ Semantic HTML
- ✅ Alt texts للصور
- ✅ Fast Loading Speed

## 🌐 النشر

### GitHub Pages
1. ارفع الملفات على GitHub
2. فعّل GitHub Pages من الإعدادات

### Netlify
1. اسحب المجلد على Netlify
2. سيتم النشر تلقائياً

### استضافة تقليدية
1. ارفع جميع الملفات عبر FTP
2. تأكد من رفع `.htaccess` للتحسينات

## 📈 نصائح للأداء
- استخدم صور WebP للأعمال
- ضغط الصور قبل الرفع
- استخدم CDN للملفات الثابتة
- فعّل HTTPS للأمان

## 📧 إعداد الفورم
الفورم يستخدم **Formspree** (أسهل طريقة):
1. افتح ملف `formspree-setup.md`
2. اتبع الخطوات (3 دقائق)
3. الرسائل توصل على إيميلك مباشرة
